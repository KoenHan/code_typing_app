# code_typing_app

## Environment
- nodejs
- npm

## Install

```
$ git clone https://github.com/KoenHan/code_typing_app.git
$ npm install
```

## How to use (Web)
<https://codetyping.herokuapp.com>

## How to use (Local)
1. Launch Server
```
$ cd /path/to/code_typing_app
$ DEBUG=first-app:* npm start
```
2. Then connect to `localhost:3000` in your web.
